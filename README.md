# Surg-R1 GitHub Pages Site

This repository hosts the public project page for **Surg-R1**.

## Local preview

Open `index.html` directly in a browser, or run any static file server.

Example with Python:

```bash
python -m http.server 8000
```

Then visit <http://localhost:8000>.

## Deployment

This repository is intended to be published with **GitHub Pages**.

- If repository name is `<username>.github.io`: publish from `main` / root.
- Otherwise, enable Pages in repository settings and publish from `main` / root.
